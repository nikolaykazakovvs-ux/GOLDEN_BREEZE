## 5. Рекомендации

### Для продакшена
1. **Уменьшить risk_per_trade до 0.5%** - снизит drawdown
2. **Добавить daily loss limit** - max -3% в день
3. **Фильтр новостей** - избегать высокой волатильности
4. **Мониторинг confidence** - алерты при падении ниже 0.90

### Дальнейшее развитие
1. **Добавить Gold features в реальном времени** (SMC, Order Blocks)
2. **Multi-timeframe analysis** - H1/H4 confirmation
3. **Sentiment integration** - новостной фильтр
4. **Ensemble** - комбинация LSTM + GBM
